Detailed Description

An abstract class for calculating the torques and the external wrenches in a Rigid Body Dynamic tree, using kinetic information (linear acceleration, angular velocity and angular acceleration of one arbitrary link, joint positions, velocities, acceleration) and FT sensor measures.
